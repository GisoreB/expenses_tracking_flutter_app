# expenses_tracking_app

A modern Flutter-based personal finance manager that helps users track their income, expenses, and financial progress with ease. 
The app uses a clean Cubit (BLoC) architecture, Hive local database, and SharedPreferences for seamless offline functionality and
persistent user preferences.
